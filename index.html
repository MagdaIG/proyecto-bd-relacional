Sistema de Gestión de Inventario

Proyecto SQL · PostgreSQL · PL/pgSQL · Validaciones · Triggers · Funciones

Descripción del Proyecto

Este proyecto consiste en el diseño y desarrollo completo de un sistema de gestión de inventario utilizando PostgreSQL. Incluye la creación de tablas, relaciones, inserciones de datos, consultas, funciones personalizadas, triggers automáticos y validaciones de integridad y consistencia.

Estructura del Proyecto

  • 01-estructura – Creación de tablas y relaciones
  • 02-insercion – Datos iniciales: productos, proveedores, transacciones
  • 03-actualizacion – Updates y deletes sobre los datos
  • 04-consultas – Consultas SQL clave: stock, historial, proveedores
  • 05-triggers – Automatización de stock con triggers
  • 06-funciones – Funciones personalizadas SQL reutilizables
  • 07-validaciones – Verificaciones de consistencia e integridad

Tecnologías Utilizadas

PostgreSQL
PL/pgSQL
SQL Avanzado
DBeaver

Habilidades Desarrolladas

  • Modelado relacional de bases de datos
  • Creación de claves primarias y foráneas
  • Uso de funciones, triggers y validaciones SQL
  • Consultas SQL complejas con joins y filtros
  • Automatización de procesos con PL/pgSQL
